FOR LOOPS

The for statement creates a loop that consists of three optional expressions, enclosed in parentheses and separated by semicolons, followed by a statement (usually a block statement) to be executed in the loop.

EXAMPLE:

for ( [ initialization ]; [ condition ]; [ final-expression ] ) statement

Within the parentheses of a for statement, follow these steps:

  • Initialize a counter (or index) variable that will be used within the loop.
  • Then, code a condition that determines when the loop will end.
  • Last, you code an expression that specifies how the counter should be incremented.
