What is a programming paridigm and why should I care?

A programming paradigm is a style of programming. There are many different programming paradigms. In this tutorial we'll learn about procedural programming, functional programming, and object-oriented programming — with a focus on object-oriented programming.

PRELIMINARY:

STATE
One of the fundamental differences between the various programming paradigms is the handling of state. State is the value of a program's variables while it is running. Global state is the value of a program's global variables while it is running.


PROCEDURAL PROGRAMMING

procedural programming is a programming style in which you write a sequence of steps moving toward a solution — each step changing the program's state (or value of the program's variables). In procedural programming, you write code to "do this, then that":

EXAMPLE:

x = 2
y = 4
z = 8
xyz = x + y + z
xyz

When you program procedurally, you store data in global variables and manipulate it with functions. Consider the following:

var f = (function (){
    var b = 10,
          c = 20,
          d = 50;
var e = function(){
    return b + c + d;
};
return e();
}());

We've taken the above code and added a var f = in front of the function. This returns the value of e() from the inner function to f and now you have a value f to be used elsewhere. Since the variables b, c, d, e are declared within the function scope of the anonymous function, we can ensure that they will not be tampered with. This is a way to do private variables in JavaScript. Now say you had 10 + 20 + 50 may places within your code. You could run this anonymous function at the beginning and substitute those occurrences with f. This idea of abstracting out code into various procedures is why it's referred to as Procedural Programming.

OBJECT ORIENTED PROGRAMMING (OOP)

The object-oriented programming paradigm addresses the problems that arise in procedural programming by eliminating global state, but instead of storing state in functions, it is stored in objects.

FOR LOOPS

The for statement creates a loop that consists of three optional expressions, enclosed in parentheses and separated by semicolons, followed by a statement (usually a block statement) to be executed in the loop.

EXAMPLE:

for ( [ initialization ]; [ condition ]; [ final-expression ] ) statement

Within the parentheses of a for statement, follow these steps:

  • Initialize a counter (or index) variable that will be used within the loop.
  • Then, code a condition that determines when the loop will end.
  • Last, you code an expression that specifies how the counter should be incremented.
